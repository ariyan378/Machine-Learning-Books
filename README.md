## ⚡ Core Curriculum Assets

This directory serves as a centralized node hosting six definitive texts bridging statistical learning, rigorous optimization, and scalable neural architectures.

* **Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow**
    * *Domain:* Pipeline engineering, tensor operations, and production framework execution.
* **An Introduction to Statistical Learning (ISL)**
    * *Domain:* Non-parametric modeling, resampling paradigms, and validation matrices.
* **The Elements of Statistical Learning (ESL)**
    * *Domain:* High-dimensional optimization, regularization theory, and formal proofs.
* **Essential Math for Data Science**
    * *Domain:* Linear algebra decompositions, multivariate calculus, and loss gradient derivation.
* **Practical Statistics for Data Scientists**
    * *Domain:* Exploratory metrics, significance testing, and rigorous experimental design.
* **The Hundred-Page Machine Learning Book**
    * *Domain:* Algorithmic synthesis and high-density structural summaries.
