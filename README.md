## ⚡ Core Curriculum Assets

This directory contains six fundamental texts bridging statistical learning, optimization, and neural architectures, directly mapping to the assets visible in `image_bcb343.png`.

*   **Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow**
    *   *Domain:* Pipeline engineering, tensor operations, and end-to-end framework execution.
*   **An Introduction to Statistical Learning (ISL)**
    *   *Domain:* Validation paradigms, resampling, and non-parametric modeling.
*   **The Elements of Statistical Learning (ESL)**
    *   *Domain:* High-dimensional optimization, regularization theory, and mathematical proofs.
*   **Essential Math for Data Science**
    *   *Domain:* Multivariate calculus, matrix operations, and loss optimization.
*   **Practical Statistics for Data Scientists**
    *   *Domain:* Exploratory metrics, significance testing, and experimental design.
*   **The Hundred-Page Machine Learning Book**
    *   *Domain:* Algorithmic synthesis and high-density architectural summaries.
